# PEHLA KADAM 2.0

PEHLA KADAM 2.0 is an AI-powered conversational adoption and growth layer for SBI YONO.  
It helps customers take their first confident financial step by guiding them in natural language, explaining recommendations, and asking for consent before any action.

## Theme Chosen
Digital Adoption

## Problem Statement
Many SBI customers have access to digital banking but do not know where to start or how to use YONO confidently. The app has powerful features, but the experience can feel complex, generic, or language-heavy for first-time users.

## Proposed Solution
PEHLA KADAM solves this by acting as a friendly digital banking assistant inside YONO. It:
- asks the user what they need,
- understands the intent,
- recommends the most relevant action,
- explains why that suggestion is shown,
- and asks for consent before proceeding.

Phase 1 focuses on helping users start.  
Phase 2 adds proactive nudges, personalized next-best actions, and a financial wellness layer to improve repeat engagement.

## Business Model / Commercial Potential
The solution can increase SBI’s commercial value by:
- improving YONO adoption,
- increasing usage of savings, RD, FD, insurance, and loan products,
- reducing support burden through guided self-service,
- and improving long-term customer engagement and retention.

## Technology Stack
This demo is built as a single-file HTML prototype using:
- HTML5
- CSS3
- JavaScript
- AI-generated UI mockup
- Browser-based screen demo

## Process Flow / Architecture
1. User opens YONO.
2. PEHLA KADAM greets the user.
3. User enters a need in text or voice.
4. The system identifies the intent.
5. The system recommends the best next step.
6. The assistant explains why this suggestion is shown.
7. The user gives consent.
8. The action is completed.
9. The user sees success feedback and next-step suggestions.
